## 🧠 Introduction

> *"Aletheia"* (Truth) — Ancient Greek for **Truth** or **Disclosure**.

We live in a world where AI can generate hyper-realistic faces, deepfake videos, and cloned voices in seconds. Misinformation spreads faster than fact-checks. **Aletheia** exists to fight that.

Built for the **Google Solution Challenge 2025**, Aletheia is a multi-modal forensic AI detection tool that combines:

- 🔬 **SightEngine** — a computer vision API trained on millions of AI-generated vs. real images
- 🤖 **Google Gemini** — a multimodal LLM that performs deep forensic visual reasoning
- ⚡ A sleek, dark-mode web interface designed for speed, clarity, and trust

Whether you're a journalist verifying a news image, a researcher studying AI media, or just a curious user — Aletheia gives you a **verdict**, not just a score.

## ⚙️ How It Works

```
User Uploads / Pastes URL
         │
         ▼
  ┌──────────────┐
  │  Frontend    │  HTML + CSS + Vanilla JS
  │  (Aletheia)  │  Drag-drop, preview, history
  └──────┬───────┘
         │  POST /api/analyze/upload (or /url)
         ▼
  ┌──────────────────────────────────────────────┐
  │              Node.js Backend                 │
  │                                              │
  │  ┌─────────────────┐  ┌──────────────────┐  │
  │  │  SightEngine    │  │  Google Gemini   │  │
  │  │  genai model    │  │  Visual Forensics│  │
  │  │  ai_generated % │  │  Multimodal LLM  │  │
  │  └────────┬────────┘  └────────┬─────────┘  │
  │           │                    │             │
  │           └────────┬───────────┘             │
  │                    │ mergeResults()           │
  │            ┌───────┴────────┐                │
  │            │  Blended Score │                │
  │            │  Verdict JSON  │                │
  │            └───────┬────────┘                │
  └────────────────────┼────────────────────────-┘
                       │
                       ▼
              Frontend renders:
              - Verdict (AI / Real / Uncertain)
              - AI Probability %
              - Confidence level
              - Estimated generator
              - AI Indicators (forensic signals)
              - Authentic Indicators
              - Technical detail paragraph
```

### Scoring Logic

| SightEngine Score | Weight | Effect |
|-------------------|--------|--------|
| ≥ 70% | 50% blend | Strong AI signal — overrides Gemini if it said "Real" |
| 40–70% | 30% blend | Moderate evidence |
| < 40% | 30% blend | Leans toward authentic |

## 🚀 Setup & Installation

### Prerequisites

```bash
node >= 18.x
npm >= 9.x
```

### 1. Clone the repository

```bash
git clone https://github.com/your-username/aletheia.git
cd aletheia
```

### 2. Install backend dependencies

```bash
npm install express cors multer axios dotenv @google/generative-ai form-data express-rate-limit
```

### 3. Configure environment variables

```bash
cp .env.example .env
```

Edit `.env`:

```env
SIGHTENGINE_API_USER=your_sightengine_user
SIGHTENGINE_API_SECRET=your_sightengine_secret
GEMINI_API_KEY=your_gemini_api_key
PORT=3001
```

> Get your keys:
> - SightEngine → [sightengine.com](https://sightengine.com)
> - Gemini → [Google AI Studio](https://aistudio.google.com/app/apikey)

### 4. Run the backend

```bash
node server.js
```

You should see:

```
🔍 Aletheia backend running on http://localhost:3001
   SightEngine: ✓ configured
   Gemini:      ✓ configured
```

### 5. Open the frontend

Simply open `index.html` in your browser, or serve it locally:

```bash
npx serve .
```
