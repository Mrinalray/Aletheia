## рҹ§  Introduction

> *"Aletheia"* (Truth) вҖ” Ancient Greek for **Truth** or **Disclosure**.

We live in a world where AI can generate hyper-realistic faces, deepfake videos, and cloned voices in seconds. Misinformation spreads faster than fact-checks. **Aletheia** exists to fight that.

Built for the **Google Solution Challenge 2025**, Aletheia is a multi-modal forensic AI detection tool that combines:

- рҹ”¬ **SightEngine** вҖ” a computer vision API trained on millions of AI-generated vs. real images
- рҹӨ– **Google Gemini** вҖ” a multimodal LLM that performs deep forensic visual reasoning
- вҡЎ A sleek, dark-mode web interface designed for speed, clarity, and trust

Whether you're a journalist verifying a news image, a researcher studying AI media, or just a curious user вҖ” Aletheia gives you a **verdict**, not just a score.

## рҹҡҖ Setup & Installation

### Prerequisites

```bash
node >= 18.x
npm >= 9.x
```

### 1. Clone the repository

```bash
git clone https://github.com/your-username/aletheia.git
cd aletheia
```

### 2. Install backend dependencies

```bash
npm install express cors multer axios dotenv @google/generative-ai form-data express-rate-limit
```

### 3. Configure environment variables

```bash
cp .env.example .env
```

Edit `.env`:

```env
SIGHTENGINE_API_USER=your_sightengine_user
SIGHTENGINE_API_SECRET=your_sightengine_secret
GEMINI_API_KEY=your_gemini_api_key
PORT=3001
```

> Get your keys:
> - SightEngine вҶ’ [sightengine.com](https://sightengine.com)
> - Gemini вҶ’ [Google AI Studio](https://aistudio.google.com/app/apikey)

### 4. Run the backend

```bash
node server.js
```

You should see:

```
рҹ”Қ Aletheia backend running on http://localhost:3001
   SightEngine: вң“ configured
   Gemini:      вң“ configured
```

### 5. Open the frontend

Simply open `index.html` in your browser, or serve it locally:

```bash
npx serve .
```
